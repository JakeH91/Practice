All Web-Dev learning resources

Here is a list of all the FREE resources I can find for teaching yourself all aspects of Web Development.

Codecademy
Basic Walkthrough Tutorials for several languages (HTML, Javascript, CSS, Ruby etc).
The Odin Project
4 Mini Courses designed to gear you up to be ready for some serious web development education. Teaches basic development, design, computer science and eventually coding.
Tuts Plus
A video guide to HTML and CSS.
Viking Code School
An in depth course that runs through ALL aspects of becoming a Web Developer.
Free Code Camp
A website that guides you through 800 hours of practice in HTML, CSS, JavaScript and jQuery, along with lessons on algorithms and general full stack web development. Followed by 800 hours worth of coding for nonprofit organisations.
Code Combat
A fun, game-based way of writing code, mainly in JavaScript.
